4) Admin Console (Time-Limited Tokens)
--------------------------------------
File:
  C:\Users\Admin\OneDrive\Documents\GUNSHOTBRO\rich-admin.html

Steps:
1. Start the RICH backend.
2. Open rich-admin.html in your browser.
3. Login:
   - Username: AMARIE
   - Password: 2007
4. Enter your API host (local or Cloudflare).
5. Choose duration (minutes/hours/days/months).
6. Click Generate token, then Copy token.
7. You can Revoke the token any time.

Admin console extras:
  - Quick Actions: start/stop server or tunnels
  - Learning Settings: enable/disable auto learning
  - File Editor: safely edit approved UI files
  - Learning Stats: view discovery metrics
  - Revoke All Tokens: wipe all temp keys
  - Action Approvals: generate auto-exec token for approved actions

Tokens auto-expire and are no longer accepted after expiry.

7) Troubleshooting
------------------
- No response:
  Check that the RICH server is running and the API host is correct.

- 401 Unauthorized:
  Wrong API key or expired token.

- Mic not working:
  The server speech backend may be missing. Install Faster-Whisper:
    .\OpenJarvis\.venv\Scripts\python.exe -m pip install faster-whisper

- Model not found:
  Ensure Ollama is installed and the model exists:
    ollama serve
    ollama pull qwen3.5:2b

Action Auto-Execute (Optional)
------------------------------
1. Open rich-admin.html and login.
2. Generate an Action Token (Action Approvals section).
3. Paste the Action Token into the RICH UI.
4. Turn on "Auto-run approved actions".
5. When RICH responds with [[ACTION:...]] it will run the action automatically.

Do NOT upload:
  - C:\Users\Admin\.openjarvis\config.toml (contains secrets)
  - OpenJarvis\ (backend code + config)
